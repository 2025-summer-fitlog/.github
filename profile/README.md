

## 1. 프로젝트 주제 및 소개
### Fitlog (Fitlog, Fitness + Log)
https://fitlog-site.vercel.app/

본 서비스는 사용자 맞춤 운동 도우미 서비스이다.

사용자의 기본적인 신체 및 운동 관련 정보를 입력받고, 사용자가 원하는 운동을 추천받을 수 있다.
또한, 사용자는 진행한 날에 대한 운동 기록을 남길 수 있는데, 운동 기록을 바탕으로 사용자에게 피드백을 제공한다.
